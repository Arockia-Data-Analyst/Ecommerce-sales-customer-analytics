# Ecommerce-sales-customer-analytics
End-to-End Data Analytics Project using Python, SQL, and Power BI
This project focuses on analyzing e-commerce sales data using Python, SQL, and Power BI to uncover business insights related to sales performance, customer behavior, profitability, and regional trends.
# TOOLS USED
• Python
• Pandas
• Matplotlib
• SQL
• Power BI
# PROJECT FEATURES
• Data Cleaning & Preprocessing
• Exploratory Data Analysis (EDA)
• SQL Business Analysis
• Interactive Power BI Dashboards
• KPI Tracking & Business Insights
